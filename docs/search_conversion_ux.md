# Natural Language Search — Conversion UX (Locked)

## Role
Natural Language Search is a **paid conversion lever**, not a convenience feature.

## Modes
### Exact Search (Free)
- Keyword based
- Manual filters (folder, tag, date)

### Natural Language Search (Paid)
- Interprets intent:
  - Time → Tag/Folder → Keyword → Domain
- Korean NL supported (80% MVP accuracy)

## Exact → NL Transition
- NL-like input is allowed for all users
- Free users see:
  - Intent preview
  - CTA: “자연어 검색으로 정확히 찾기 (Pro)”

## Failure UX
- No errors
- Fallback to Exact Search
- Suggestions:
  - Date filter
  - Tag filter

## Success Signal
- Show “why this result appeared”
- Make difference **visible**, not explained
